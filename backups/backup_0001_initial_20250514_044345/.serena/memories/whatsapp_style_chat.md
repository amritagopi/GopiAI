# Стилизация чата в стиле WhatsApp

В этом обновлении реализован чат в стиле WhatsApp с современным и интуитивно понятным интерфейсом.

## Изменения
1. **Внешний вид сообщений**
   - Созданы "пузыри" для сообщений с разным цветом для пользователя и ассистента
   - Добавлена адаптивная высота для текстовых сообщений
   - Применен фон в стиле WhatsApp (#e5ddd5)

2. **Поле ввода сообщений**
   - Создано расширяемое поле ввода, которое увеличивается по мере ввода текста
   - Реализовано ограничение высоты (35-100px) с вертикальной прокруткой при превышении лимита
   - Добавлена обработка Enter для отправки и Shift+Enter для новой строки

3. **Кнопки и функциональность**
   - Добавлены кнопки для вставки эмодзи, прикрепления файлов и изображений слева
   - Реализована умная кнопка микрофона/отправки, которая меняется при наличии текста в поле ввода
   - Интегрирована работа с диалогом эмодзи и выбором файлов

4. **Стилизация**
   - Создан кастомный CSS для полной стилизации компонентов чата
   - Добавлены стили для полосы прокрутки, кнопок и пузырей сообщений
   - Реализованы интерактивные эффекты при наведении и нажатии

## Новые классы
1. `MessageBubble` - пузыри для отображения сообщений
2. `ChatHistoryArea` - область прокрутки для истории сообщений
3. `ExpandingTextEdit` - расширяемое поле ввода текста

## Особенности реализации
- Полностью переработана структура ChatWidget для более современного интерфейса
- Добавлена поддержка прикрепления файлов и изображений с предварительным просмотром
- Исправлен обработчик событий для корректной работы с клавишами Enter/Shift+Enter
- Стиль полностью соответствует дизайну WhatsApp для привычного пользовательского опыта
