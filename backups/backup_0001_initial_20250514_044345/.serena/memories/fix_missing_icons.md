# Исправление отображения иконок в приложении

После обновления системы иконок были внесены дополнительные изменения для обеспечения корректного отображения всех иконок:

1. Обновлен `app/ui/icon_adapter.py`: 
   - Добавлены маппинги недостающих иконок для типов файлов (js_file, html_file, css_file, pdf, и т.д.)
   - Добавлен fallback для файловых иконок (если нет точного соответствия, но имя оканчивается на _file)

2. Обновлен `app/ui/code_editor_widget.py`:
   - Добавлена поддержка иконок для вкладок редактора кода
   - Добавлен метод _get_file_icon_name для получения иконки на основе расширения файла
   - Обновлен метод add_new_tab для установки иконок на вкладки

3. Обновлен `app/ui/main_window.py`:
   - Восстановлено меню "Icons" в меню Help
   - Обновлены методы _setup_help_menu и _show_icon_info для работы с Lucide иконками
   - Добавлены категории иконок для удобного отображения списка

Эти изменения обеспечивают:
- Корректное отображение иконок файлов в Project Explorer
- Отображение иконок на вкладках редактора кода на основе типа файла
- Доступ к просмотру всех доступных иконок через меню Help -> Icons
