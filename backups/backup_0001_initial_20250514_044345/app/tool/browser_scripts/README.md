# Браузерные JavaScript-скрипты

Этот пакет содержит набор JavaScript-скриптов для взаимодействия с веб-страницами через браузер, встроенный в приложение GopiAI.

## Структура пакета

Скрипты организованы по категориям в отдельных модулях:

- **page_info.py** - скрипты для получения основной информации о странице
- **content.py** - скрипты для работы с текстовым и структурным содержимым
- **elements.py** - скрипты для работы с DOM-элементами
- **forms.py** - скрипты для заполнения и отправки форм
- **monitoring.py** - скрипты для мониторинга изменений страницы

## Доступные скрипты

### Информация о странице

- **GET_PAGE_INFO** - получает основную информацию о странице: заголовок, URL, мета-теги, заголовки H1/H2, ссылки и изображения

### Содержимое страницы

- **GET_PAGE_TEXT_CONTENT** - извлекает текстовое содержимое страницы с разбивкой на параграфы
- **GET_NAVIGATION_MENU** - находит навигационные меню на странице
- **GET_TABLE_DATA** - извлекает данные из таблиц
- **GET_STRUCTURED_DATA** - находит структурированные данные (JSON-LD, микроданные)

### Работа с элементами

- **GET_ELEMENTS_BY_SELECTOR** - находит элементы по CSS-селектору и извлекает информацию о них
- **CLICK_ELEMENT** - выполняет клик по указанному элементу с проверкой видимости

### Работа с формами

- **FILL_FORM** - заполняет поля формы указанными значениями (поддерживает текстовые поля, select, checkbox и radio)

### Мониторинг страницы

- **MONITOR_PAGE_CHANGES** - запускает непрерывный мониторинг изменений на странице
- **GET_PAGE_PERFORMANCE** - получает метрики производительности страницы

## Использование в инструментах

Эти скрипты используются инструментами браузера для выполнения различных действий:

```python
from app.tool.browser_scripts import GET_PAGE_INFO

# Пример использования скрипта в инструменте
current_tab.execute_javascript(GET_PAGE_INFO, callback)
```

## Расширение функциональности

Вы можете добавить новые скрипты, следуя этим рекомендациям:

1. Поместите скрипт в соответствующий модуль по категории
2. Добавьте импорт в `__init__.py`
3. Добавьте константу для скрипта в `__all__`
4. Используйте скрипт в инструментах браузера через `execute_javascript`
