# Интеграция браузерных инструментов с UI в GopiAI

## Кратко

Все инструменты браузера (`BrowserControlTool`, `BrowserAnalyzerTool`, `BrowserScreenshotTool`) теперь отправляют уведомления о ходе работы, ошибках и результатах в UI через сигнал `progress_update` у `MultiBrowserWidget`. Это позволяет отображать статус операций в статус-баре главного окна.

## Как это работает

- Каждый инструмент реализует метод `notify_ui`, который вызывает `self.browser_widget.progress_update.emit(message)`.
- В методах `execute` всех инструментов при каждом возврате результата или ошибке вызывается `notify_ui` с соответствующим сообщением.
- В `MainWindow` сигнал `progress_update` подключён к методу обновления строки состояния.

## Пример

```python
# Внутри execute любого browser_tool
self.notify_ui("Переход на https://example.com")
self.notify_ui("Ошибка: Активная вкладка не найдена")
```

## Важно
- Если сигнал не подключён в UI, уведомления не будут видны.
- Для новых инструментов реализуйте аналогичный метод уведомлений.

## Для разработчиков
- Следуйте архитектурным принципам: минимальная связанность, сигналы/слоты для коммуникации.
- Все ошибки и успехи должны быть видны пользователю.
