# Система иконок GopiAI

## Архитектура
- Все иконки хранятся в `assets/icons/` в формате SVG.
- Манифест `assets/icons/manifest.json` содержит сопоставление логических имён иконок и файлов.
- Для доступа к иконкам используется синглтон `IconManager` (`app/ui/icon_manager.py`).
- Глобальные функции `get_icon(name)` и `list_icons()` обеспечивают удобный доступ к иконкам по имени.
- В случае отсутствия иконки возвращается fallback или системная стандартная иконка.

## Интеграция с UI
- Все основные UI-компоненты используют `get_icon()` для установки иконок на кнопки, меню, вкладки и т.д.
- Для новых функций и инструментов рекомендуется добавлять новые иконки в манифест и SVG-файлы в папку `assets/icons/`.

## Рекомендации
- При добавлении новых иконок обязательно обновлять `manifest.json`.
- Для тестирования корректности отображения иконок использовать автоматические тесты (см. ниже).

## Тестирование
- Планируется добавить автоматические тесты для проверки наличия всех иконок из манифеста и корректной загрузки через `IconManager`.
