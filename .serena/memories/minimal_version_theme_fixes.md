# Исправления тем и интерфейса для минимальной версии

В минимальной версии приложения были выполнены следующие исправления:

1. **Изменения в theme_selector.py**:
   - Исправлен класс ThemeManager для корректной работы с темами
   - Реализована загрузка и сохранение настроек тем в файл
   - Создан метод _get_style_sheet_for_theme, не использующий CSS переменные (которые вызывали ошибки парсинга)

2. **Изменения в complementary_theme_dialog.py**:
   - Увеличена ширина кнопок (с 110px до 130px) для корректного отображения текста
   - Уменьшены боковые отступы (padding с 15px до 10px) 
   - Изменен текст кнопки "Применить" на более короткий "Сохранить"
   - Добавлен метод _apply_dialog_style для установки непрозрачного фона диалога
   - Увеличены отступы в нижней части диалога для визуального улучшения

3. **Изменения в minimal_app.py**:
   - Исправлена функция base64_to_pixmap для более надежной конвертации base64 в иконки
   - Обновлена функция apply_global_theme для применения стилей через ThemeManager
   - Добавлена резервная таблица стилей на случай проблем с ThemeManager

4. **Изменения в icons.py**:
   - Добавлена функция get_icon_pixmap для совместимости с обновленным кодом

Основные проблемы, которые были решены:
- Некорректное отображение кнопок в диалоге с обрезанным текстом
- Прозрачный фон диалога настройки темы
- Некорректное применение цветовых настроек к элементам интерфейса
- Ошибки при загрузке иконок из base64

Сохранение настроек темы теперь происходит в файл `minimal_version/settings/current_theme.json` и автоматически загружается при запуске приложения.
