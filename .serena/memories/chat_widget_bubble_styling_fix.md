# Исправление проблемы стилизации пузырей сообщений в чате

## Проблема
В интерфейсе чата пузыри сообщений ассистента имели белый фон (#ffffff), из-за чего надписи и иконки (тоже светлые) плохо читались на светлом фоне.

## Изменения

1. **Изменен цвет фона пузырей сообщений ассистента**:
   - Заменен белый фон (#ffffff) на более нейтральный светло-серый (#ececec)
   - Добавлен явный темный цвет текста (#363636) для обеспечения контрастности

2. **Согласованность стилей пользовательских сообщений**:
   - Добавлены аналогичные стили для QTextBrowser в пузырях пользователя
   - Установлен темный цвет текста (#303030) и прозрачный фон

## Технические детали
Изменения были сделаны в файле `app/ui/widgets.py` в классе `MessageBubble`:

```python
# Для сообщений ассистента
self.setStyleSheet("""
    #assistantMessageBubble {
        background-color: #ececec;
        border-radius: 10px;
        margin: 2px 80px 2px 10px;
    }
    
    #assistantMessageBubble QTextBrowser {
        color: #363636;
        background-color: transparent;
    }
""")

# Для сообщений пользователя
self.setStyleSheet("""
    #userMessageBubble {
        background-color: #dcf8c6;
        border-radius: 10px;
        margin: 2px 40px 2px 80px;
    }
    
    #userMessageBubble QTextBrowser {
        color: #303030;
        background-color: transparent;
    }
""")
```

## Результат
- Текст в пузырях сообщений ассистента стал хорошо читаемым на светло-сером фоне
- Сохранена общая стилистика оформления в стиле WhatsApp
- Обеспечена согласованность стилей между пузырями пользователя и ассистента
