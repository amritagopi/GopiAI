# Анализ проекта GopiAI

## Общая информация
GopiAI — это персональный AI-ассистент, созданный как альтернатива коммерческим подпискам. Проект использует PySide6 (Qt для Python) для создания пользовательского интерфейса.

## Ключевые компоненты
1. **Интерфейс пользователя (UI)**:
   - Основан на PySide6 с поддержкой тем (светлая/тёмная)
   - Многоязычность (русский/английский)
   - Модульная структура с виджетами для различных функций
   - Система иконок с поддержкой SVG и различных наборов иконок

2. **Функциональность**:
   - Чат с агентами (MCP - Model Context Protocol)
   - Поиск через DuckDuckGo
   - Sequential thinking (пошаговое мышление)
   - Загрузка и обработка файлов
   - Поддержка tools для вызова внешних функций
   - Встроенный браузер (BrowserAgent)
   - Поддержка vision-моделей
   - Генерация изображений (через Together.ai)

3. **Архитектура**:
   - Модульная структура с четким разделением логики
   - Паттерн Singleton для менеджеров (темы, иконки, переводы)
   - Event-driven архитектура на основе сигналов Qt
