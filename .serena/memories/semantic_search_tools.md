# Инструменты семантического поиска по коду GopiAI

В этом проекте реализованы специальные инструменты для быстрого и эффективного семантического поиска по кодовой базе, которые следует использовать вместо обычного поиска по файлам.

## Основные компоненты:

1. **index/code_search.py** - инструмент семантического поиска, использующий Llama Index
2. **index/dependency_explorer.py** - инструмент для анализа зависимостей между файлами
3. **scripts/semantic_search.py** - удобный интерфейс к обоим инструментам

## Примеры использования:

### Семантический поиск по кодовой базе:
```
python scripts/semantic_search.py search "как работает авторизация"
```

### Анализ зависимостей по имени файла:
```
python scripts/semantic_search.py deps --search "auth"
```

### Анализ зависимостей конкретного файла:
```
python scripts/semantic_search.py deps --file "app/auth/login.py"
```

## Преимущества:
- Быстрый поиск без зависаний
- Семантический поиск вместо точного соответствия
- Структурированный анализ зависимостей
- Более эффективное использование времени

## Важно!
При анализе кода всегда сначала использовать эти инструменты вместо обычного grep-поиска или чтения всех файлов подряд. Это значительно ускоряет процесс работы с кодовой базой и позволяет быстрее найти нужные компоненты.
