# Оптимизация структуры проекта GopiAI

## Выполненная работа

Проект GopiAI был реорганизован для улучшения структуры и упрощения навигации. Скрипты были перемещены из корневой директории в соответствующие тематические поддиректории.

## Новая структура директорий

1. **scripts/run/** - скрипты запуска приложений и демонстраций
   - Содержит все скрипты с префиксом `run_*`

2. **scripts/setup/** - скрипты настройки и инициализации
   - Содержит скрипты с префиксами `setup_*` и `init_*`

3. **scripts/ui/** - скрипты для работы с UI, иконками и темами
   - Содержит скрипты для работы с стилями, темами и ресурсами

4. **scripts/mcp/** - скрипты для работы с MCP
   - Содержит скрипты и файлы, относящиеся к MCP

5. **scripts/utils/** - утилиты и вспомогательные скрипты
   - Содержит различные скрипты для очистки, диагностики и других задач

6. **project_health/analyzers/** - скрипты для анализа кода и проекта
   - Содержит инструменты для проверки кода, поиска неиспользуемых файлов, анализа зависимостей

7. **project_health/reports/** - директория для отчетов
   - Содержит отчеты и скрипты для их создания

## Созданные инструменты

1. **scripts/optimize_project_structure.py** - скрипт для оптимизации структуры проекта
   - Автоматически определяет и перемещает скрипты в соответствующие директории
   - Поддерживает режим предварительного просмотра (dry-run)
   - Включает обновление импортов (отключено из-за сложности)

## Преимущества новой структуры

1. Улучшенная организация кода по функциональным группам
2. Чистый корневой каталог проекта
3. Простая навигация по типам скриптов
4. Логичное разделение инструментов анализа и утилит
5. Упорядоченное хранение отчетов в отдельной директории

## Важные файлы, оставленные в корне проекта

1. main.py - основной скрипт запуска
2. setup.py - скрипт установки
3. requirements.txt - зависимости проекта
4. Конфигурационные файлы (.gitignore, pyproject.toml и т.д.)

## Использование семантического поиска

Для эффективной работы с проектом рекомендуется использовать инструменты семантического поиска:

```
python scripts/semantic_search.py search "ваш поисковый запрос"
python scripts/semantic_search.py deps --search "имя_файла"
python scripts/semantic_search.py deps --file "путь/к/файлу"
```
