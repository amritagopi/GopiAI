# Использование системы иконок в UI GopiAI

## Общие принципы
- Для всех кнопок, меню, вкладок и других элементов интерфейса используются иконки только через функцию `get_icon(name)` из `app/ui/icon_manager.py`.
- Прямое указание путей к SVG/PNG-файлам запрещено — только логические имена из манифеста.
- При добавлении новых функций или инструментов обязательно добавлять иконку в манифест и использовать её через `get_icon`.

## Примеры
```python
from app.ui.icon_manager import get_icon
button = QPushButton(get_icon("play"), "Запустить")
menu_action = QAction(get_icon("settings"), "Настройки", self)
```

## Рефакторинг существующего кода
- Все прямые пути к иконкам (например, через QIcon("assets/icons/xxx.svg")) должны быть заменены на `get_icon("xxx")`.
- Если иконка отсутствует в манифесте, добавить её и обновить `manifest.json`.

## Проверка
- Для проверки корректности использования иконок запускать тесты из `tests/test_icon_manager.py`.
