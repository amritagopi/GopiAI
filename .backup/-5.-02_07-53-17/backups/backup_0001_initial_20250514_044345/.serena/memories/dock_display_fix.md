# Исправление отображения элементов интерфейса

## Проблемы
При восстановлении приложения из файла main_window.py.backup были обнаружены несколько проблем с отображением элементов интерфейса:

1. В методе `_apply_initial_layout` отсутствовала настройка видимости и размеров доков
2. Отсутствовал вызов функции `fix_duplicate_docks`
3. В файле `docks.py` использовался неправильный подход к скрытию заголовка для `chat_dock`

## Внесенные изменения

### 1. Обновление метода _apply_initial_layout в main_window_core.py
- Добавлена явная настройка видимости доков
- Добавлена настройка размеров доков (20% ширины для боковых панелей)
- Добавлен вызов функции `fix_duplicate_docks`

### 2. Исправление настройки заголовка chat_dock в docks.py
- Удален специфический код для скрытия заголовка (empty_title_widget и CSS)
- Заменен на стандартное применение заголовка через `apply_custom_title_bar`

## Результаты
После внесенных изменений:
- Доки отображаются правильно с правильными заголовками
- Интерфейс теперь выглядит согласованно, доки имеют правильные размеры и положение
