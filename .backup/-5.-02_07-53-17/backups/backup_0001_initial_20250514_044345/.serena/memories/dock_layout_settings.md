# Настройки расположения панелей в GopiAI

## Оптимальные размеры панелей

В MainWindow были внесены следующие изменения для оптимального расположения панелей:

1. Ширина боковых панелей (проводник проекта и чат) уменьшена с 50% до 20% от ширины окна, чтобы оставить больше места для центрального редактора кода.

2. Эти настройки применяются в двух методах:
   - `_apply_initial_layout()` - при первом запуске приложения
   - `reset_dock_layout()` - при сбросе расположения панелей пользователем

## Отображение бинарных файлов

Для корректного отображения файлов с нестандартным форматом:

1. В классе `CodeEditor` добавлена поддержка определения типа файла:
   - По расширению (для известных бинарных типов)
   - По содержимому (если содержит непечатаемые символы)

2. При обнаружении бинарного файла:
   - Файл открывается в режиме шестнадцатеричного просмотра
   - Редактор переключается в режим только для чтения
   - К заголовку вкладки добавляется пометка "(Hex)"

3. Шестнадцатеричное представление включает:
   - Адрес каждой строки
   - Шестнадцатеричное представление байтов
   - ASCII представление символов (непечатаемые заменяются точками)

Эти настройки позволяют более эффективно использовать пространство окна и корректно работать с различными типами файлов.
