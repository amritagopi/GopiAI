# Система иконок GopiAI

## Архитектура
- Все иконки хранятся в `assets/icons/lucide/` в формате SVG.
- Используется библиотека [Lucide](https://lucide.dev/) для минималистичных масштабируемых SVG-иконок.
- Для доступа к иконкам используется синглтон `LucideIconManager` (`app/ui/lucide_icon_manager.py`).
- Для обратной совместимости используется `IconAdapter` (`app/ui/icon_adapter.py`).
- Глобальные функции `get_icon(name)` из `icon_adapter.py` обеспечивают удобный доступ к иконкам по имени.
- Поддерживается настройка цвета и размера иконок.
- В случае отсутствия иконки возвращается fallback или системная стандартная иконка.

## Интеграция с UI
- Все основные UI-компоненты используют `get_icon()` для установки иконок на кнопки, меню, вкладки и т.д.
- Для новых функций и инструментов рекомендуется использовать существующие иконки из библиотеки Lucide или добавлять новые SVG-файлы в папку `assets/icons/lucide/`.

## Преимущества новой системы
- Единый современный стиль иконок с Lucide
- Масштабируемые SVG-иконки без потери качества
- Поддержка тем (изменение цвета иконок)
- Более простое добавление новых иконок

## Тестирование
- Для тестирования иконок можно использовать утилиту `test_lucide_icons.py`.
