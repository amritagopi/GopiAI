# Улучшения интерфейса минимальной версии

## Исправленные проблемы

1. **Увеличен размер главного окна**
   - Установлен размер по умолчанию 1024x768 вместо минимально допустимого 800x600
   - Добавлен явный вызов `resize()` при инициализации

2. **Устранена прозрачность диалога выбора темы**
   - Отключен атрибут `Qt.WA_TranslucentBackground`
   - Добавлены явные фоновые цвета для всех компонентов диалога

3. **Улучшено отображение кнопок**
   - Увеличены размеры кнопок (min-width: 140px, min-height: 40px)
   - Установлен фиксированный размер кнопок (150x45) в диалоге цветовой темы
   - Увеличены отступы между кнопками (spacing: 20px)
   - Увеличен размер шрифта (font-size: 16px)

4. **Добавлено цветовое различие между кнопками**
   - "Сохранить" - зеленая (#4CAF50)
   - "OK" - синяя (#2196F3)
   - "Отмена" - красная (#F44336)

5. **Исправлены стили CSS**
   - Заменены переменные CSS (@border_color, @titlebar_background) на конкретные цвета
   - Добавлены отдельные идентификаторы для кнопок (#saveButton, #okButton, #cancelButton)
   - Добавлены состояния hover и pressed для всех кнопок

## Измененные файлы
- minimal_app.py
- complementary_theme_dialog.py
- theme_selector.py

## Рекомендации на будущее

- Добавить поддержку мультиязычности для текста на кнопках
- Рассмотреть возможность сохранения размеров окна в настройках
