# Создание нового файла

Фича создания нового файла использует следующие компоненты:

1. В MainWindow метод `_new_file()`:
   - Создаёт новый экземпляр CodeEditor
   - Вызывает editor.set_new_file()
   - Добавляет новую вкладку "Untitled" в central_tabs
   - Устанавливает фокус на новый редактор

2. В CodeEditor метод `set_new_file()`:
   - Очищает текст редактора
   - Устанавливает file_path в None
   - Сбрасывает флаг модификации документа
   - Обновляет заголовок вкладки, если редактор находится в контейнере вкладок

3. Иконка "Новый файл" на панели инструментов:
   - В toolbar.py добавляется действие main_window.new_file_action
   - В menu_manager.py это действие подключается к методу main_window._new_file
   
Проблема была в том, что метод set_new_file() отсутствовал в классе CodeEditor, хотя вызывался из MainWindow._new_file().

Если функция создания нового файла не работает, следует проверить:
1. Наличие и корректность метода set_new_file() в CodeEditor
2. Правильность подключения сигналов в menu_manager.py
3. Видимость и доступность иконки "Новый файл" в панели инструментов
4. Наличие central_tabs в MainWindow
