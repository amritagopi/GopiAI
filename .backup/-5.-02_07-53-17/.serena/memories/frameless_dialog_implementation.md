# Реализация безрамочных диалоговых окон

В рамках улучшения пользовательского интерфейса приложения были внесены изменения для создания безрамочных диалоговых окон с закругленными углами.

## Реализованные изменения:

1. Сделан безрамочным диалог настройки цветовой темы (ComplementaryThemeDialog)
   - Добавлен флаг Qt.FramelessWindowHint
   - Реализована обработка перетаскивания окна мышью
   - Добавлен кастомный заголовок с кнопкой закрытия
   - Применены стили CSS для закругленных углов

2. Улучшен алгоритм движения маркеров в цветовом круге
   - Реализован более плавный и предсказуемый алгоритм определения типа движения
   - Добавлен гистерезис для предотвращения резких переключений между режимами
   - Улучшена отзывчивость перемещения

3. Улучшена логика работы с пользовательскими темами
   - Реализовано сохранение и восстановление последней примененной темы
   - Улучшен диалог перезаписи существующей темы (добавлен выбор: перезаписать/создать новую/отмена)
   - Автоматическая загрузка последней использованной темы при старте

## Задачи для будущих улучшений:

1. Создать базовый класс FramelessDialog, который будет содержать общую логику безрамочного окна
2. Сделать все диалоговые окна приложения безрамочными и с закругленными углами:
   - Диалоги сообщений (QMessageBox)
   - Диалоги ввода (QInputDialog)
   - Диалоги выбора файлов (QFileDialog)
   - Другие диалоги

3. Реализовать единый стиль для всех диалоговых окон
4. Добавить анимацию появления и скрытия диалогов для лучшего пользовательского опыта

Все будущие диалоговые окна должны создаваться с использованием нового базового класса для обеспечения единого внешнего вида и поведения.
