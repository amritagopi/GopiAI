# Плагин закругленных углов и безрамочных окон

## Описание
Плагин добавляет закругленные углы ко всем окнам и элементам интерфейса и делает все окна безрамочными.

## Возможности
1. Автоматически делает все окна и диалоги безрамочными
2. Добавляет закругленные углы к элементам интерфейса (кнопки, поля ввода, меню)
3. Позволяет перетаскивать окна кликом в любом месте
4. Перехватывает создание новых окон и автоматически применяет к ним стили
5. Применяется ко всему приложению без изменения исходного кода

## Технические детали
- Использует фильтр событий для перехвата создания новых окон
- Переопределяет обработчики событий мыши для реализации перетаскивания
- Применяет стили CSS для оформления
- Сохраняет состояние обработанных окон

## Настройка
Параметры плагина можно настроить через методы:
- `set_border_radius()` - для элементов интерфейса
- `set_window_radius()` - для окон
- `set_button_radius()` - для кнопок

## Интеграция
Плагин интегрируется через систему хуков:
- Реагирует на хук `after_stylesheet_applied`
- Реагирует на хук `window_created`
- Устанавливает глобальный фильтр событий

## Установка
Плагин устанавливается автоматически при запуске приложения через `app.py`.
