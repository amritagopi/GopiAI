# Улучшения интерфейса чата

В интерфейсе чата были исправлены две основные проблемы:

## 1. Оптимизация поля ввода сообщений

### Проблема
Поле ввода (ExpandingTextEdit) имело избыточную начальную высоту и некорректное поведение при изменении контента.

### Решение
- Установлена минимальная начальная высота (32px)
- Добавлена логика изменения высоты в зависимости от наличия текста:
  - При пустом поле - минимальная высота
  - При наличии текста - адаптивная высота, соответствующая контенту
- Включена вертикальная полоса прокрутки при превышении максимальной высоты (150px)
- Оптимизирован метод `_adjust_height()` для корректного изменения размеров

## 2. Устранение дублирующихся заголовков

### Проблема
Отображались два заголовка: от dock-виджета ("Chat") и из ChatWidget ("Чат").

### Решение
- Улучшен метод скрытия заголовка dock-виджета
- Создан пустой виджет с нулевой высотой для полного скрытия стандартного заголовка
- Оставлен только кастомный заголовок внутри ChatWidget

## Технические детали
- Модификация класса `ExpandingTextEdit` в файле `app/ui/widgets.py`
- Изменение способа скрытия заголовка dock-виджета в файле `app/ui/docks.py`
- Сохранены все функциональные возможности с улучшением UI/UX
