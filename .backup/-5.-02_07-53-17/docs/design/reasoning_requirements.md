# Требования для Reasoning Agent

## Необходимые компоненты

Reasoning Agent полагается на два основных компонента, которые являются **обязательными** для его работы:

1. **Sequential Thinking** - модуль структурированного мышления, используемый агентом для анализа проблем, разделения их на последовательность шагов и принятия решений.

2. **Serena Integration** - модуль взаимодействия с файловой системой, поиском в коде и работой с памятью.

## Поведение при отсутствии компонентов

* Без модуля **Sequential Thinking** агент не может:
  - Формировать логические цепочки рассуждений
  - Создавать планы действий
  - Анализировать проблемы в структурированном виде
  - Оценивать разные варианты решений

* Без модуля **Serena Integration** агент не может:
  - Работать с файловой системой проекта
  - Искать символы в коде
  - Анализировать структуру кода
  - Сохранять и загружать файлы памяти

## Проверки наличия компонентов

Агент выполняет следующие проверки:

1. **При инициализации** - проверяет наличие необходимых инструментов в MCP.
2. **При каждом запуске** - дополнительно проверяет доступность модулей через их методы `is_available()`.
3. **При вызове инструментов** - проверяет корректность работы соответствующих модулей.

## Сообщения об ошибках

При отсутствии необходимых модулей агент выдает подробные сообщения с указанием того, какие именно компоненты недоступны:

- `Missing required reasoning tools: [список инструментов]`
- `Sequential Thinking module is not available. Cannot continue execution.`
- `Serena module is not available. Cannot continue execution.`
- `Failed to initialize Sequential Thinking module`
- `Failed to initialize Serena module`

## Тестирование

Для проверки поведения агента при отсутствии модулей используйте тест:

```bash
python test_reasoning_agent.py --test-missing-modules
```

## Зависимости

Для работы обоих модулей требуется:

1. Настроенный и запущенный MCP-сервер
2. Правильно сконфигурированный проект GopiAI
3. Библиотеки MCP, установленные в окружении
