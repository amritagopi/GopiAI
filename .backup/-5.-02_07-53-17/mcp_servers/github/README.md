# GitHub MCP Server

MCP сервер для GitHub API, позволяющий выполнять операции с файлами, управлять репозиториями, осуществлять поиск и многое другое.

## Установка

### Требования
- Docker
- GitHub Personal Access Token с необходимыми разрешениями

### Сборка Docker образа
```bash
cd mcp_servers/github
docker build -t mcp/github .
```

### Настройка
1. Создайте GitHub Personal Access Token с разрешениями `repo` (или `public_repo` для публичных репозиториев)
2. Добавьте конфигурацию MCP сервера в файл настроек Cline

## Использование

Сервер предоставляет следующие инструменты:
- Создание и обновление файлов
- Отправка нескольких файлов в одном коммите
- Поиск репозиториев
- Создание репозиториев
- Получение содержимого файлов
- Создание issues и pull requests
- Форк репозиториев
- Создание веток
- Работа с issues и pull requests
- Поиск кода, issues и пользователей

Подробная документация по каждому инструменту доступна в исходном README проекта.
