# Инструкции для автоматической работы с GopiAI

Это руководство для AI-ассистента Cursor, работающего с проектом GopiAI.

## Контекст проекта

GopiAI - это приложение на Python с использованием PySide6, где:
- Основная творческая работа завершена
- Требуется подключить функции к интерфейсу
- Используется MVC-архитектура
- Необходимо минимизировать человеческое вмешательство

## Приоритеты разработки

1. Интеграция UI и бэкенда (app/ui/main_window.py)
2. Браузерные компоненты (app/tool/browser_* и app/ui/browser_*)
3. Система настроек для конфигурации агентов
4. Интернационализация (app/ui/i18n/)
5. Система иконок

## Инструкции для автономной работы

При каждом запуске автоматически:
1. Прочитай файл .serena/memories/workflow_state.md для понимания текущего состояния
2. Изучи .serena/memories/development_plan.md для понимания общего плана
3. Проверь .serena/memories/component_tasks.md для конкретных задач
4. Следуй правилам в .cursorrules
5. Работай автономно, минимизируя запросы к пользователю
6. После выполнения задачи обнови workflow_state.md

## Автоматические действия

- Используй Sequential Thinking для разбиения сложных задач
- Документируй код и изменения
- Следуй архитектурным принципам MVC
- Придерживайся стиля кода PEP 8
