# Реализация интерфейса GopiAI

В соответствии с требованиями по интерфейсу, в проекте GopiAI были внесены следующие изменения:

## Добавленные компоненты:

1. **app/ui/dock_title_bar.py** - класс для создания кастомизированных заголовков окон с кнопками:
   - Закрытие (для всех окон)
   - Сворачивание (для браузера)
   - Разворачивание/Восстановление (для браузера)

2. Обновление интерфейса в **app/ui/main_window.py**:
   - Добавлены методы для настройки ограничений доков
   - Реализованы стили для разных типов вкладок (текстовые, терминал, браузер)
   - Добавлена поддержка обновления кастомных заголовков при смене темы

## Основные элементы интерфейса:

1. **Файловый проводник**:
   - Закреплен слева, не может быть отсоединен
   - Может изменять размер вправо, но всегда остается слева
   - Имеет кнопку закрытия

2. **ИИ-чат**:
   - Закреплен справа, не может быть отсоединен
   - Может изменять размер влево, но всегда остается справа
   - Имеет кнопку закрытия

3. **Терминал**:
   - Закреплен внизу и слева
   - Можно изменять размер, но нельзя отсоединять или перемещать
   - Имеет вкладочную структуру
   - Вкладки имеют отличительный цвет (зеленоватый оттенок)
   - Имеет кнопку закрытия

4. **Текстовый редактор**:
   - Закреплен сверху и слева
   - Изменяет размер в обоих направлениях
   - Имеет вкладки со стандартным оформлением

5. **Браузер**:
   - Может быть плавающим или закрепленным
   - Имеет вкладки с синеватым оттенком
   - Имеет полный набор кнопок (закрытие, сворачивание, разворачивание)

## Дополнительные особенности:

1. **Адаптация к темам**:
   - Поддержка светлой и темной темы для всех элементов интерфейса
   - Автоматическое обновление при смене темы

2. **Кнопки управления**:
   - Единый стиль для всех кнопок
   - Кнопки закрытия подсвечиваются красным при наведении

3. **Размеры и пропорции**:
   - Минимальные и максимальные размеры для каждого элемента
   - Начальный макет с оптимальным распределением пространства

## Размещение окон:

Все окна (кроме файлового проводника и чата) могут:
- Перемещаться (становиться плавающими)
- Быть прикреплены к верхней части (рядом с текстовым редактором)
- Быть прикреплены к нижней части (рядом с терминалом)

При этом логика вкладок сохраняется во всех случаях.
