# Анализ проекта GopiAI

## Текущее состояние проекта

### Готовые модули:
- **Основной UI-фреймворк**: Реализован базовый интерфейс с поддержкой тем (светлая/темная)
- **Многоязычность**: Базовая поддержка русского и английского языков
- **Агентская система**: Основа для агентов создана (базовые классы и абстракции)

### Требуют доработки:
- **Взаимодействие UI и бэкенда**: Недостаточно реализовано взаимодействие между UI-компонентами и агентами
- **Браузерные компоненты**: Новые файлы в `app/tool/browser_*` и `app/ui/browser_*` требуют интеграции
- **Система настроек**: Компоненты для сохранения/загрузки конфигурации агентов
- **Интернационализация**: Файлы переводов в `app/ui/i18n/` требуют доработки
- **Система иконок**: Интеграция неотслеживаемых иконок в ресурсы проекта

## Архитектура проекта
- MVC-подобная архитектура:
  - Агенты - модели
  - UI-компоненты - представления
  - Инструменты (tools) - часть контроллера

## Зависимости
- PySide6 для GUI
- OpenAI, MCP для работы с LLM
- Инструменты для поиска (DuckDuckGo, Google, Baidu)
- Браузерные инструменты (browser-use, playwright)
- Pocketflow Framework
- Рабочие утилиты (Docker, FastAPI, uvicorn)

## Процесс разработки
- Git для контроля версий
- Минимальное тестирование
- Скрипты для компиляции ресурсов и запуска
