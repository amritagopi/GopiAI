# Changelog

All notable changes to GopiAI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-09-09

### Added
- Initial release of GopiAI
- CrewAI integration for multi-agent orchestration
- Qt-based user interface with PySide6
- Intelligent AI model rotation system
- Support for Gemini, OpenAI, and other AI providers
- Production-ready Gunicorn WSGI server
- Cross-platform packaging with Briefcase
- Comprehensive logging and error handling
- Dark/Light theme support
- Real-time chat interface
- Agent specialization and task distribution
- Rate limit handling and automatic model switching

### Technical Features
- Python 3.10+ support
- Linux AppImage packaging
- Windows MSI installer support
- macOS DMG package support
- Modular architecture with separate UI and server components
- REST API communication between UI and CrewAI server
- Comprehensive test suite
- Production deployment configuration

### Supported Platforms
- Linux (Ubuntu 20.04+, Linux Mint 22+)
- Windows 10+
- macOS 11+ (Big Sur or later)
