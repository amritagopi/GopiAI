# Предпочтение русского языка

## Языковые настройки
- **Основной язык общения**: Русский
- **Документация**: Предпочтительно на русском языке
- **Комментарии в коде**: На русском языке где это уместно
- **Техническая терминология**: Использовать русские эквиваленты когда они существуют, в скобках указывать английские термины при первом упоминании

## Принципы коммуникации
- Использовать полные, детальные объяснения на русском языке
- Не упрощать технические концепции - переводить их корректно
- Сохранять профессиональный уровень технической документации
- При необходимости использования английских терминов - давать пояснения на русском

## Исключения
- Названия переменных, функций, классов в коде остаются на английском (стандарт программирования)
- Официальные названия технологий, библиотек, фреймворков
- Команды терминала и API endpoints
- Конфигурационные файлы где требуется английский синтаксис
